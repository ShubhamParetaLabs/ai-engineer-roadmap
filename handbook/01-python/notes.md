# Python kya hai? / What is Python?
-> Python is a high-level, interpreted, general-purpose programming language that focuses on readability and developer productivity.

# Python AI me kyu use hoti hai? / Why Python for AI?
-> Python is easy to learn, easy to read, has a huge ecosystem of AI libraries (like TensorFlow, PyTorch), and supports rapid application development.

# Python ke advantages kya hain? / Advantages of Python
-> Simple syntax, huge community support, open-source nature, and versatility across domains.

# Python kaha use hoti hai? / Where is Python used?
-> Artificial Intelligence, Machine Learning, Data Science, Web Development (Backend), Automation, and Scripting.

## Explanation of Key Features:

1. Interpreted Language:
-> Iska matlab hai ki code directly line-by-line execute hota hai, ek baari me poora compile nahi karna padta. Isse code test aur debug karna bohot easy ho jata hai.

2. Automatic Memory Management:
-> Python khud variables ke liye memory allocate aur delete (Garbage Collection) karta hai. C ya C++ ki tarah aapko manually memory manage nahi karni padti.

3. Extensible & Embeddable:
-> Extensible: Aap chaho to Python ke andar C ya C++ ka code use kar sakte ho (fast processing ke liye).
-> Embeddable: Aap apne dusre (e.g. C/C++) programs ke andar Python code daal sakte ho, taaki script chalayi ja sake.

4. Versatile (All-rounder):
-> Python sirf ek cheez ke liye nahi bani. Ise Web Development, AI/Machine Learning, Data Science, Automation (Scripts) aur Hacking/Security me bhi efficiently use kiya jata hai.
